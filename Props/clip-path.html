clip-path

clip-path は、要素のクリッピング領域(どの部分を切り抜くか)を指定するプロパティです。初期値は none です。

clip-path を試す

clip-path: path('M 0 200 L 0,75 A 5,5 0,0,1 150,75 L 200 200 z');
clip-path: path('M 0 200 L 0,110 A 110,90 0,0,1 240,100 L 200 340 z');
Dan FadorによるPixabayからの画像

補足

このプロパティに使用される値は basic-shape のデータ型の値になります。
